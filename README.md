# Как пользоваться гитом

Гитом лучше всего пользоваться через командную строку, поэтому открываем ее через Win + R и вводим cmd
Далее вводим путь куда мы хотим поместить наш проект:

```shell
cd <путь куда вы хотите подложить проект>
```

### Импортируйте 

далее сделайте:

```shell
git clone https://github.com/SonLetGo/MountinComplex.git
```

чтобы скопировать репозитрий

### Работа

Откройте проект 1/1.csproj в Visual Studio

Далее когда вы сделали что хотели (добавили форму или изменили) оправьте изменения для остальные через интерфейс:
```shell
Вкладка Git -> Отправить -> Название изменения ->  Зафиксировать все и отправить
```

### Получить изменения 

Для получения что сделаи другие надо:
```shell
Вкладка Git -> Получить
```
